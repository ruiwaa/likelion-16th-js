# 4. 터미널에 값 출력

//콘솔(Console)
//-콘솔! 기록해줘(값)의 의미
//console.log(값)

## 1) 숫자 값(number) 출력
console.log(9);
-> 출력값: 9

## 2) 문자 값(string) 출력
   ```
    console.log("자바스크립트 프로그램(작은 따옴표)");
    ->출력값: 자바스크립트 프로그램(작은 따옴표)

    console.log("자바스크립트 프로그램(큰 따옴표)");
    ->출력값:자바스크립트 프로그램(큰 따옴표)

    console.log(`자바스크립트 프로그램(역 따옴포)`);
    ->출력값:자바스크립트 프로그램(작은 따옴표)

    console.log(`자바스크립트 프로그램(역 따옴포)`);
    ->출력값:자바스크립트 프로그램(역 따옴포)
```

## 3) 논리값 출력
console.log(false);
console.log(true);
->출력값:
false
true

# 7. 콘솔 패널에 오류 또는 경고 출력

브라우저 개발자 도구에 있는 탭 중 하나로, 자바스크립트 실행 결과와 디버깅 정보를 확인하는 공간이다.

// 콘솔 오류를 나타낼 때
// console.error();

//콘솔 경고를 나타낼 때
// console.warn();

>예시
console.error("❌ 문자 값은 따옴표로 묶어 출력하지 않으면 오류입니다.");
console.warn("⚠️ 자바스크립트의 데이터를 기억하려면 변수 또는 상수가 필요합니다.");

# 8. 콘솔 그룹화 출력
// console.group(그룹화할 값)
console.groupEnd();
-> 그룹화로 묶어줄 경우, 반드시 "console.groupEnd()"로 맺음지어야 한다.

>예시) 숫자(정수)를 그룹화할 경우
// 숫자 (정수)
console.group('숫자 (정수)')
console.log(10)
console.log(101)
console.log(908)
console.groupEnd();
출력값: 
숫자(정수)
10
101
908
