# 4. 터미널에 값 출력

//콘솔(Console)
//-콘솔! 기록해줘(값)의 의미
//console.log(값)

## 1) 숫자 값(number) 출력
console.log(9);
-> 출력값: 9

## 2) 문자 값(string) 출력
    console.log("자바스크립트 프로그램(작은 따옴표)");
    ->출력값: 자바스크립트 프로그램(작은 따옴표)

    console.log("자바스크립트 프로그램(큰 따옴표)");
    ->출력값:자바스크립트 프로그램(큰 따옴표)

    console.log(`자바스크립트 프로그램(역 따옴포)`);
    ->출력값:자바스크립트 프로그램(작은 따옴표)

    console.log(`자바스크립트 프로그램(역 따옴포)`);
    ->출력값:자바스크립트 프로그램(역 따옴포)

## 3)논리값 출력
console.log(false);
console.log(true);
->출력값:
false
true
