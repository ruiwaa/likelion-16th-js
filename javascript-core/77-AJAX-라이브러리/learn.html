Fetch API vs Axios 라이브러리

라이브러리가 필요한 이유 (Fetch의 한계)

Fetch API는 저수준(Low-level) API로, 사용 시 다음과 같은 번거로움이 있습니다.

  • 중복 호출: .then()을 두 번 사용해야 데이터 파싱 가능.
  • 수동 파싱: Content-Type에 따라 .json(), .text() 등을 직접 분기해야 함.
  • 오류 처리: 404, 500 에러도 성공으로 간주하므로 response.ok를 수동으로 체크해야 함.

Axios 라이브러리의 장점

가장 대중적인 AJAX 라이브러리로, Fetch의 단점을 해결하고 코드를 간결하게 만듭니다.
